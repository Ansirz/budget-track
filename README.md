BUDGET TRACK
Description:
 Simple offline expense tracker made with flutter.
Features:
-  Persistence of transactions, achieved with Hive.
-  Pie chart and bar chart for analytics
-  Clear all transactions button
-  Dropdown to choose different categories
What I learned:
-  How to add charts for analytics using the fl_chart library
-  How to perform operations with values in the hive database
-  How to create data models and use them in Hive
-  How to make bottom nav
Tech stack:
- Flutter
- Dart
